# QT入门

## 一.weight窗口

### 1.1 窗口设计

|                          |                  |                |
| ------------------------ | ---------------- | -------------- |
| setWindowTitle("string") | 设置窗口标题     | 窗口标题的名称 |
| resize(200,200)          | 设置窗口大小     | 长度，宽度     |
| setFixedSize(300,400)    | 设置固定窗口大小 | 长度，宽度     |
|                          |                  |                |

### 3.1 对话框

#### 3.1.1 模态对话框

阻塞时创建对话框，在操作时无法对其他对话框进行操作。

#### 3.1.2 非模态对话框

非阻塞式对话框，在操作时可以对其他对话框进行操作。
